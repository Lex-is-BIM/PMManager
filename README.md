**Плагин для расширенного управления свойствами проекта и материалами в Renga.**

Плагин предоставляет расширенные возможности по управлению свойствами проекта и материалами, включая их удаление. Все операции выполняются выборочно, что позволяет точно контролировать изменения в проекте.

---

## 🚀 Возможности

### 📋 Управление свойствами проекта
- ✅ Удаление выбранных свойств
- 📤 Экспорт свойств в файл JSON
- 📥 Импорт свойств из файла JSON

### 🧱 Управление материалами
- 🗑️ Удаление выбранных материалов
- 🗑️ Удаление выбранных многослойных материалов

---

## ⚙️ Операции со свойствами

| Операция | Описание |
|----------|----------|
| **Удалить свойства** | Удаляет выбранные свойства из проекта. Список исключений определяет свойства, не выделяемые по умолчанию. |
| **Экспортировать свойства** | Сохраняет выбранные свойства в JSON-файл. |
| **Импортировать свойства** | Загружает отсутствующие свойства из JSON-файла. |
| **Настройка списка исключений** | Управление перечнем исключаемых свойств. |

---

## 🧱 Операции с материалами

| Операция | Описание |
|----------|----------|
| **Удалить материалы** | Вызывает диалог выбора с двумя вкладками: *Материалы* и *Многослойные материалы*. |
| **Настройка списка исключений** | Управление перечнем исключаемых материалов. |

---

## 🛡️ Список исключений

- Обеспечивает определение свойств, материалов и многослойных материалов, **не выделяемых по умолчанию** при удалении
- **Автоматически заполняется** при открытии проекта
- Позволяет защитить критически важные элементы от случайного удаления

---

## 🖱️ Контекстное меню

Для всех списков доступно:
- ✅ **Выделить все элементы**
- ❌ **Снять выделение со всех элементов**

## 📥 Установка

1. Скачайте архив с плагином
2. Распакуйте содержимое в папку:
   ```
   %APPDATA%\Renga\Plugins\
   ```
3. Запустите Renga
4. Плагин появится в меню **"Управление"** или на панели инструментов
